# tmux

## sessions

```
# create
tmux new-session -s <session-name>

# attach
tmux attach -t <session-name>

# create or attach
tmux new -s <session-name> || tmux a -t <session-name>

# list sessions
tmux list-sessions
tmux ls
```

- `attach -d`: attach and detach any other sessions
- `attach -r`: attach in read-only mode

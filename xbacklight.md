# xbacklight

Adjust backlight using randr

```bash
# set backlight to 50 %
xbacklight -set 50

# increase backlight 5 %
xbacklight -inc 5

# decrease backlight 5 %
xbacklight -dec 5
```


## Fading options

- `-time N`: Fade for N ms (default: 200)
- `-steps N`: Fade in N steps (default: 20)
